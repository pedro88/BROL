# S02: Demande à la communauté pour emprunt — UAT

**Milestone:** M014
**Written:** 2026-05-14T07:12:40.182Z

1. Aller sur /requests → affiche la liste des demandes ouvertes (ou vides si aucune)
2. Cliquer sur "Nouvelle demande" → le dialog s'ouvre
3. Remplir titre, description optionnelle, zone → créer → la demande apparaît dans la liste
4. Vérifier le status badge (Ouverte)
5. En tant qu'owner, voir la demande et cliquer "Répondre à cette demande" → marque comme fulfill
6. Vérifier que la demande apparaît maintenant avec le status "Traitée"
