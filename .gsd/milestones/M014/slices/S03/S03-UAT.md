# S03: Système de notification — UAT

**Milestone:** M014
**Written:** 2026-05-14T11:37:02.830Z

1. Faire un prêt → une notification RETURN_REMINDER est créée pour l'emprunteur
2. Laisser un avis sur quelqu'un → notification REVIEW_RECEIVED pour la cible
3. Répondre à une demande communauté → notification REQUEST_FULFILLED pour l'auteur
4. Aller sur /notifications → voir la liste des notifications avec badges de statut
5. Cliquer sur une notification → la marquer comme lue
6. Cliquer "Tout marquer lu" → toutes les notifications passent en lu
