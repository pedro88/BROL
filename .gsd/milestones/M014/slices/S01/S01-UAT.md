# S01: Profil utilisateur avec notes et commentaires — UAT

**Milestone:** M014
**Written:** 2026-05-14T06:55:35.243Z

1. Naviguer vers /profile/[userId] → affiche les infos de l'utilisateur
2. Vérifier que les badges s'affichent s'il en a
3. Vérifier que la note moyenne et le nombre de commentaires s'affichent
4. Vérifier que la liste des commentaires se charge
5. En tant qu'utilisateur éligible (après un prêt terminé avec cette personne), cliquer sur "Laisser un avis" → le dialog s'ouvre
6. Sélectionner une note (1-5 étoiles), ajouter un commentaire optionnel, soumettre → l'avis apparaît dans la liste
7. Vérifier que le badge "Critique" est awardé après avoir laissé un avis
8. Vérifier qu'on ne peut pas se noter soi-même ni noter deux fois le même échange
